# m-pagination

基于 react 的分页器组件

React Pagination Component

# Development
```shell
git pull https://github.com/tantaiMoon/m-pagination.git

npm install
# OR
yarn install

npm start
```

# Example

# Usage
```js
import Pagination from 'm-pagination';
ReactDOM.render(<Pagination />, container);
```

# API

|Parameter|Description|Type|Default|
|---|---|---|---|
